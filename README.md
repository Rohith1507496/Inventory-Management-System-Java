# Inventory Management System

## Overview
A console-based Inventory Management System developed using Java.

## Features
- View Products
- Add Product
- Search Product
- Update Product
- Delete Product
- Restock Product
- Low Stock Alert
- Inventory Summary

## Technologies Used
- Java
- OOP Concepts
- ArrayList
- Collections Framework

## How to Run

Compile:

```bash
javac *.java
```

Run:

```bash
java Main
```
